Course Description

Essential skills, concepts and the use of virtual cameras in 2D and 3D animation environments: planning, pre-production, storyboards and pre-visualization using real-world camera principles is combined with rendering, sound and post-production in a nonlinear environment.

Course Competencies

  • Compare and contrast the visualization process in traditional arts vs. directing a shot in a digital environment. (I)
  • Manipulate aperture, depth of field, lenses, and attributes needed to control a virtual camera in various environments. (I-II)
  • Create conceptual sketches, plans and storyboards for a project. (III)
  • Apply lighting and compositional concepts to create a mood in a digital environment. (IV)
  • Determine appropriate camera attributes and shot flow using plans and storyboards. (IV-V)
  • Apply shot flow and directing techniques to a project. (V)
  • Stage a scene for a character and/or product shot. (VI)
  • Apply the appropriate lines of action to virtual cameras in various environments. (VII)
  • Select the appropriate sound for the shot. (VIII)
  • Render a project using appropriate project specifications and output methods. (IX-X)
  • Use post-production techniques to edit in a nonlinear environment. (X)
