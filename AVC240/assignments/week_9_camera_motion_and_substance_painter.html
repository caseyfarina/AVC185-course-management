Project Objective: Students will learn about camera movement terminology. Students will construct a reel that showcases cinematic uses of a moving camera.

  • Course Competencies
    • Compare and contrast the visualization process in traditional arts vs. directing a shot in a
      digital environment. (I)
    • Manipulate aperture, depth of field, lenses, and attributes needed to control a virtual
      camera in various environments. (I-II)
    • Apply lighting and compositional concepts to create a mood in a digital environment. (IV)
    • Apply shot flow and directing techniques to a project. (V)
    • Stage a scene for a character and/or product shot. (VI)
    • Select appropriate sound for the shot. (VIII)
    • Render a project using appropriate project specifications and output methods. (IX-X)
    • Use post-production techniques to edit in a nonlinear environment. (X)

Project

  • Create a showcase dance music video using Redshift (preferred) or Viewport Renderer with the following shots
    • Repaint one of the Mixamo Models in Substance Painter
    • Use dance motion capture and character models from mixamo 
    • Create a dynamic dance environment using lights, staging, set design
    • Give the shots time to breath, the whip pan and zoom should have time on either end of the camera animation.
    • Label each shot with a lower third
    • Add a sound track before laying out the clips on your timeline. Edit to the music.
    • Make compelling images! What do is your visual language?
      • Whip Pan 
        • from one character to another
        • Use linear blur in AE to enhance the effect
        • time remap if necessary
        • use ease keyframes
      • Crash Zoom Shot
        • Animate the focal length only in a short amount of time
        • Use the Radial Blur in AE to enhance the Zoom blur effect
        • use ease keyframes
      • Tilt shot
        • Animate the tilt axis of the camera
        • use linear keyframes
      • Push shot
        • Animate the forward movement of the camera
        • use linear keyframes
      • Pull shot
        • Animate the backward movement of the camera
        • use linear keyframes
      • Roll
        • Animate the roll axis of the camera
        • use linear keyframes
