The Principles of Motion Design will guide our animation throughout the semester.

Project Objective: Students will prototype a game with animation, interactivity, physics simulation and user interface elements. 

Course Objectives

    • Use principles of animation
    • Create non-linear experiences
    • Create and use advanced buttons for user input
    • Manipulate text and related objects, variables and expressions
    • Create movie clips, navigate timelines

Project: 

Use the same project from week one. Just create new scenes in that project. The scene must contain the following

  • Please install Unity 6 if you have not already
  • Download this rolling ball starter template and be sure to open in Unity 6
    • interactionTemplate.zip
    • Open the "ballPlayer" scene
    • This template contains the trigger script and the restart script
  • Prototype a rolling ball game that include the following 
    • 4 zones
      • Start, physics obstacle, animated obstacle, Finish
    • At least 1 substance from substance community
      • Sign in with your Maricopa credentials to access substance community 
    • At least one substance from substance sampler
    • Use the restart script to allow the player to restart after falling off
    • Use the trigger script to send a winning message to the player if they complete the level

    • Less that 4 minutes
    • Play through the level and demonstrate the obstacles and win message
    • What went well?
    • What was difficult?
    • How did you select and design the materials?
