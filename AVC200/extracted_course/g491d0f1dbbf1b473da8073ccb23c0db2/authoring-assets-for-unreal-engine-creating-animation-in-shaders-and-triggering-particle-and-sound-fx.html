Project Objective: Students will design, model, and texture game ready 3D assets

  • Course Competencies
    • Navigate Program Interface and tools.
    • Use principles of animation
    • Create non-linear experiences
    • Create movie clips, navigate timelines
    • Optimize performance
    • Distribute various output formats

Project Requirements:

  • 1st person Unreal Engine Game Requirements
    • Create at least 2 new game assets for walls and ceilings that support the game world and narrative
      • Consider light and shadow when designing these assets. How could the assets cast shadow to enhance the immersive nature of the game?
    • Add a particle effect and sound effect for the pickup 
    • Store the pickup value in a new blueprint and print the number of pickups acquired to the screen
    • Create a new game prop that uses an animated texture in Unreal engine
    • Model an additional  object for your game using the High Poly Sculpt, Low Poly bake workflow (coffee bean).
    • Draw a map on paper of the level as it is currently designed.
    • Consider the map and draw in a different color the changes you want to make to the layout of the first person level. 

Deliverable:

  • Canvas
    • An OBS recording of a tour of your Unreal level. Address the following questions during your discussion.
      • limit to 4 minutes
      • Present all of the new objects in the game
      • Discuss how you used the Hight Poly - Low Poly - Bake workflow
      • Display the material and shader graph for the animated material in Unreal engine. How did you  use the animated texture?
      • Show your map (take a picture to display on the screen) and discuss how you will expand the level design
