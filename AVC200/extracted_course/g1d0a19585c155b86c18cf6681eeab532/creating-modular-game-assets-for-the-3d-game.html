  • Set the Unity editor snap settings to 0.5 x 0.5 y 0.5 z

Create the following modular game assets. Create textures for each element using Substance Painter. Feel free to create other assets as well. 

  • Platform
  • Stairs with separate collider mesh
  • Vertical Environment Asset
  • Short Environment Asset
  • Gateway Environment Asset

Import all of the of the meshes and textures into Unity. Create new materials for each asset and arrange all of your assets (and floor tiles) into a section of a level.
