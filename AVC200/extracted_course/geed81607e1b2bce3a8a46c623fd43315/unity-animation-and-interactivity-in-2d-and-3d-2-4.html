For our first Mini-Project we'll create a 2D interactive bouncing ball and a 3D interactive bouncing ball. 

Workflow

  1. Download Unity Hub and Unity 2021.3 LTS
  2. Create a ball image in Adobe Illustrator
    1. 500 pixels x 500 pixels
    2. 72 dpi
    3. RGB
  3. Import the image as a 2D sprite into Unity
  4. Animate the bouncing ball with gravity. The animation should exhibit deceleration on the way up and acceleration on the way down
  5. Use the _clickEventOnCollider.cs  script (right click and download) to make the ball bounce on mouse click.
  6. Create a 3D sphere that also animates upon mouse click
  7. Install the Unity Recorder package to create an mp4 of your functioning application.
