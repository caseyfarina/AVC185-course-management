Final Game Design Requirements: Follow your game design document and create a compelling 3D interactive experience

  1. Rigged Character
    1. Make specific creative decisions about the movement speed, jump height, animation playback speed. 
    2. Animations 
      1. Walk/Run
      2. Idle
      3. Jump
      4. Connected to the Animator for Transitions
  2. Three zones
    1. Each zone should be incrementally completed
      1. Gate each zone
        1. Door
        2. Bridge
        3. Wall - door script
    2. Different mechanisms for each zone
      1. Key
      2. Collectible 
      3. Platforming - tagged as "moving platform"
    3. Three different mesh objects 
      1. Modeled in Blender
        1. UV unwrapped
      2. Painted in Substance Painter
    4. One Animated Environmental mesh
      1. Create a dynamic environment by using the Principles of Motion Graphics
    5. At least camera cut scene
      1. Include switching from the 3rd person follow to another camera 
      2. Can include additional Ui messages and environmental triggers
  3. Use Interface
    1. TextMeshPro Objects
      1. Need to convert a TTF or OTF to a TextMeshPro Font Asset in order to use in TMP
        1. Create -  TextMeshPro - Font Asset - SDF
        2.  
      2. Select a font that matches the style of the game
        1. Serif for older styles
        2. Sans Serif for newer styles
      3. Make a Design decision about the color of the text
    2. Required user interface elements
      1. Instructions/title at the beginning of the game
      2. Feedback about collectibles
      3. Incremental instructions about game objects
        1. On/Off Trigger Enter/trigger exit events
      4. Win Message - let the player the have complete the game
      5. UI Scaling requirements
        1. 16:9 aspect ratio
        2. Canvas is set to Overlay, and Scale with Screen Size
    3. Game VFX
      1. Three different particle effects in the game
        1. One should be triggered by gameplay
        2. Two can be static environmental FX
    4. Sound/Music
      1. Background Music
        1. Win Music
        2. Zone Music
      2. Sound FX
        1. Three different sound FX triggered in the game
        2. For example "collectible pickup"
        3. Environmental sounds
    5. Build the game as an executable - EXE file
      1. Build the game as a Windows Executable
      2. Make sure the scene that contains the game is the only scene in the build settings
      3. Name the game in the Player Settings in the Build Settings - "Product Name"
      4. Include the "escape to quit" script
        1. _escapeToQuit.cs
          1. On a game object in the game
        2. Build the game
          1. Do not build to the project folder
          2. Once the build is complete, select all the contents of the build folder and ZIP
          3. DO NOT UPLOAD ZIP TO DISCORD -
