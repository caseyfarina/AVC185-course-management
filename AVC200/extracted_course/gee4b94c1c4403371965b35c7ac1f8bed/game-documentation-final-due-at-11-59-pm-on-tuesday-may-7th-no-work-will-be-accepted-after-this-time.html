Project Objective: Students will create a portfolio of their game design work using Behance, Unity Recorder, Unreal Engine and SketchFab.

  • Course Competencies
    • Create storyboards, develop concepts, visual style
    • Use principles of animation
    • Create non-linear experiences
    • Create and use advanced buttons for user input
    • Manipulate text and related objects, variables and expressions
    • Use sound in production
    • Optimize performance
    • Distribute various output formats

Project Requirements:
