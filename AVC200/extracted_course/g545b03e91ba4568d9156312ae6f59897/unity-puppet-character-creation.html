Puppet Character Workflow

  1. Sketch Character
  • Conceptualizing where the character will move: articulating parts
  • Put each articulating part on it’s own layer
  • Save both AI and export a reference PNG
  • Save as “exploded version,” then move each articulating component to a distinct area of the square sprite sheet. 
    • 1000 x 1000
    • 72 dpi
    • DISCTRETE RECTANGULAR BOUNDING BOX FOR EACH COMPONENT
  • Import the reference PNG and the exploded sprite sheet into UNITY
  • Set the sprite mode to MULTIPLE from SINGLE - and APPLY
  • Open the sprite in the “sprite editor” 
  • Select SLICE and automatic - APPLY
  • Select each slice and rename, and change pivot points - APPLY
  • Create an empty game object to hold the entire character.
  • Bring the sprites into the scene and arrange them in the correct hierarchy under the main game object
  • Create new “Animation” and drag it onto the top of the character hierarchy
  • Make sure the top of the character heirarchy is selected and use the animation window to create the idle animation loop
