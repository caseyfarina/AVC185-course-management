Introduction

We need to become familiar with animation in the aptly named Edge Animate. For those of you that have experience with other animation tools (Cinema 4D, After Effects) you'll find that Edge Animate functions in much the same way.

Project

Use your forest project as a starting point to add some animated elements to the forest environment. Feel free to make changes to the forest scene to facilitate the animation requirement below and to improve the look and "feel" of your scene. Make your scene BUILD onto the screen. End with the title. 

  • A symbol that animates along a path
  • A symbol that animates using a nonlinear mode of easing for position.
  • A symbol that animates using a nonlinear mode of easing for scale.
  • A symbol that animates using a nonlinear mode of easing for rotation.
  • 3 symbols that have looping animation.
  • A symbol that has animated image filter properties.

Deliverable

Please turn in a ZIP file of your entire project folder and a 200 word document that explains how your scene meets the requirements of the project. Also discuss any problems you had and ideas for elements that you would like to include in the future. PLEASE DO NOT SUBMIT WPS FILES FOR YOUR DOCUMENT. Acceptable file formats are .doc, .rtf, .txt, .docx.

  • ZIP of your project folder
  • Word doc discussing how your forest project meets the requirements of the project
