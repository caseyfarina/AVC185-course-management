This week you animate, light, and add materials to your 3D scene.

Your scene must meet the following requirements.

  1. At least 2 different objects (multiple copies of one object count as ONE object) should be animated using the Animator with Animation Clips. 
  2. At least 2 object should be animated using the 3D physics system. 
    • Stationary objects need to have a 3D collider component
    • Moving object need to have a 3D collider component and 3D rigidbody component
  3. At least 3 objects should use a script for animation.  I am compiling all class scripts on this page.
    • The linear movement script will move an object at a stead pace.
  4. Use the simpleSpawner script in your scene to spawn new objects at least once.
  5. Use either point lights, spot lights, or a directional light to light your scene (No more than 10, point lights or spot lights)
  6. Use the Global Volume to create a post processing stack for your scene
  7. Use materials and textures (polyhaven.com) to create the look and feel of your scene.
  8. If you are creating an outside scene use an HDRI to create reflections in your scene.
