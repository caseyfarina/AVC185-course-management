Please Zip your built game and your project folder for the interactive map prototype that includes the following. 

  • House
  • Animated figure in a walk cycle ( you must create the sprite sheet, do not download a sprite sheet.)
  • Tree
  • Bush
  • Car
  • Fence
  • Background (grass,sidewalk, etc)

All of the sprites besides the background should exhibit the following level of interactivity.

  • Color changes onMouseOver
  • Size changes slightly onMouseOver
  • Label appears onMouseOver
