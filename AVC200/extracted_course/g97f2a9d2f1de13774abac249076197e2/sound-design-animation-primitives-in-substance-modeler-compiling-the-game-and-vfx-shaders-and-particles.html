Project Objective: Students will polish and build a 3D game.

Course Objectives

    • Create non-linear experiences
    • Create and use advanced buttons for user input
    • Manipulate text and related objects, variables and expressions
    • Create movie clips, navigate timelines

Project: 

Build the rolling ball game, zip the game, post to your google drive and share your game on discord.

Game Requirements

  • Sound
    • Ambient Sound
    • Dynamically Triggered Music track
    • Jump sound effect
  • VFX
    • Use or create an animated shader using shader graph (water, lava, etc)
    • Create 3 different particle effects to use in the game
  • Environment Design
    • >= (greater than or equal to) 4 objects modeled in Blender
    • > 3 objects modeled in Substance Modeler
      • use both the "clay" and the "primitive" workflows
    •  > 3 objects painted in Substance Painter
    • > 2 animated environment objects
    • A single object can "count" in multiple categories. Please note these are minimums, design your best game.
  • UI
    • Welcome Message
    • Win Message
    • Fall Message
    • Apply the "Escape to Quit" script to a blank game object to make it easier to quit the game
      • escapeToQuitNewInput.cs
    • restart the game script
  • Game Design
    • >= four distinct zones
    • reuse modular assets and prefabs to create a coherent world
    • Apply the squash and stretch animation principle to at least one animated game object
    • Adjust the Free look camera settings to fit the game design
    • Add a large "fall trigger" that prompts the user to restart the game
  • Build the game
    • Build the game to a folder that is NOT in the project folder
    • apply a build setting to select either windowed play or fullscreen play
    • Zip the entire build folder, not just the exe
    • Post the zip file to your google drive
    • Set the permissions on the google drive to "anyone with link"
    • post the link to discord
