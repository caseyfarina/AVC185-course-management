Project Objective: Import your completed character into Unity from Blender

  • Export all the animations as an FBX from Blender
    • Idle
    • Run
    • Jump
  • Export all of the textures from Substance Painter using the Unity (universal render pipeline) setting.
    • export as an sbsar file
  • Download the starter unity project and import the textures and animations
