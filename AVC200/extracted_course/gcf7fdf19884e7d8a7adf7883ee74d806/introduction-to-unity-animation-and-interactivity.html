Instruction:

Animating in Unity, Introduction To Unity, Using Unity Recorder

  • https://www.youtube.com/watch?v=AREv6uFcmxE

Project Objectives: Students will become acquainted with the Unity authoring environment and create a bouncing ball animation. 

Course Objectives:

  • Navigate Program Interface and tools.
  • Use animation techniques to create linear experiences
  • Use principles of animation
  • Create non-linear experiences
  • Manipulate text and related objects, variables and expressions
  • Create movie clips, navigate timelines
  • Distribute various output formats

Project Workflow:

Download Unity Hub and use Unity Hub to Download Unity 2022.3 . If the numbers and letters after the .3 are slightly different, that is OK, you should proceed.

  • https://unity.com/download

Download the template Unity project for this assignment.

  • AVC200_bounceBallTemplate.zip

You should then unzip this folder and open the project in 2022.3. I go through this process in the tutorial below. I was using 2021.3 last year, so adjust accordingly. The process for opening a project is exactly the same.

  • https://youtu.be/61Gpu4Nwh74
  • If you receive a "Not a valid Project" error when opening the project. Simply move down one level in the folder tree. You need to tell Unity to open the folder that is ONE level above the ASSETS folder. Not two. 
  • You should click on the bouncingBallScene in the Scenes folder immediately after you open the project. You want to do your work in that scene.
