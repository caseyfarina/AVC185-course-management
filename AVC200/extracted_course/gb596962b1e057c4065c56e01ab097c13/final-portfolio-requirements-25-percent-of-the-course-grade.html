Project Objective: Students will present their work in a digital portfolio. 

  • Course Competencies
    • Navigate Program Interface and tools.
    • Use principles of animation
    • Create non-linear experiences
    • Create movie clips, navigate timelines
    • Optimize performance
    • Distribute various output formats

Project Requirements:

  • Portfolio page requirements. DO NOT place the portfolio requirements in this order. Put the description first but organize according to what makes sense for your game. 
    • >= 300 word game description. Describe the design of the game and how you used specific software tools to realize your artistic vision. 
    • At least 3 game meshes of your design on Sketchfab
      • Describe the process of creating each game mesh. Focus on artistic design and technical implementation.
    • At least three different process images, for example
      • Level Sketch
      • Substance Painter or Blender process screen shots
      • Unity or Unreal shader graphs, code, components or blueprints
      • Add a 100 word description of each process image
    • At least 3 short video clips in gif format, Add a 200 word description for each short video clip. Describe the process of creating these animations and particle effects. 
      • Particle Effects
      • Door animation
      • Shader animation
    • A longer video clip (not gif format) of gameplay
    • A link to the downloadable game

Deliverable:

  • Canvas
    • A presentation of your completed portfolio. Present each portfolio page and discuss the following
      • What new did you learn this semester?
      • How will you use this work when applying for jobs?
      • How will you use what you learned in this class in your own personal art practice? 
      • What specifically will you work on this summer to continue your work in Animation and Interactivity?
    • The link to the Unity Behance page
    • The link to the Unreal Behance page
  • Discord
    • Please post the links to your Behance pages. Comment on ALL the other Behance portfolios.
