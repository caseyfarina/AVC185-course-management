Project Objective: Students will design materials using a variety of source materials.

  • Course Competencies
    • Navigate Program Interface and tools.
    • Use principles of animation
    • Create non-linear experiences
    • Create movie clips, navigate timelines
    • Optimize performance
    • Distribute various output formats

Project Requirements:

  • Design materials using Substance Sampler
    • Please don't copy my process exactly, creatively apply the principles you've learned.
      • use the stone wall image as a  base to create a substance
      • use the stone wall to create a substance with flooded water
      • use the stone wall to create a substance with moss and/or dirt
      • use firefly to create an image using AI that you convert into a substance
      • take at least two pictures and use them to create substances
      • make a metal substance from scratch
    • Import all of the substances into your Unreal level. Be sure to set the substance graph to "tri planar" on all the imported substances.
      • Use all of the imported substances in your Unreal game

Deliverable:

  • Canvas
    • A zip file containing the following
      • An OBS recording of a tour of your Unreal level. Address the following questions during your discussion.
        • Limit to < five  minutes
        • Show all the substances you designed in Substance Sampler
        • Ask any questions you have about Substance Sampler, show substance Sampler in the recording so the question is more detailed and specific.
        • What ideas did this project give you for the future? 
        • Show at least three other aspects of your level that you adjusted based on this week's feedback
  • Discord
    • Post your screen recording and comment on at least three other student posts with the following prompt
      • Make a suggestion about another substance that could enhance the visual environment of the game.
