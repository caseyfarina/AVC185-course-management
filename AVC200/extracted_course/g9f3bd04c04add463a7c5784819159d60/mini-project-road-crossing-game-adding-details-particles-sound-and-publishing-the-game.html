Update the Game Manager

You'll need this update to the avoidance game manager. Notice that the name is different. It will not conflict with the old gameManager.

  1. Import into Unity
  2. Remove the previous game manager script from the game manager game object
  3. Add the "gameManagerComplete" script to the game manager game object
  4. Add the appropriate game objects to the new game manager script.

Adding an obstacle and second enemy to the game

  • You need to create an obstacle sprite. Perhaps a rock, fence, or barrier of some kind.
  • Add a second vehicle or enemy of your own design. It must have sprite animation (wheels turning, or equivalent) as well as animation across the screen.
  • There is no tutorial for adding a second enemy to the game. Review the previous enemy design if necessary.

https://youtu.be/hk0T8Rog2Hg
