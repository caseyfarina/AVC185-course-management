Project Objective: Students will implement real time VFX, sound, music, and build a third person 3D game.

  • Course Competencies
    • Navigate Program Interface and tools.
    • Use principles of animation
    • Create non-linear experiences
    • Create movie clips, navigate timelines
    • Optimize performance
    • Distribute various output formats

Project Requirements:

  • Create an Epic Games account
    • https://store.epicgames.com/en-US/
  • Download and Install Unreal  Engine 5.4
  • Set up a new project using the 3rd person template (pay attention to the settings discussed in the instruction)
  • Create a vertical level design that uses elevators to traverse vertically in the space.
    • Animate a bouncing ball in your level
      • Gravity
      • Squash and stretch
    • Apply materials from the standard assets
    •  Use at least four different objects from Fab in the level design. Resuse the objects throughout the level in a way that makes sense in your world.
    • Create a glowing material to emit light into the scene
    • Create Point Lights, Spot lights, and adjust the Directional light to create a dynamic looking environment.
    • Use a particle system in a manner that makes sense for your level design
