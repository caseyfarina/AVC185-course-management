Then follow these principles for converting your C4D meshes and models for use in Unity.

  • Organize the mesh to facilitate animation. Group objects with ALT + G 
  • Set the anchor point for each group. The shortcut for the anchor point tool is L .THIS IS A TOGGLE IN C4D! Be sure to turn the anchor point tool OFF after adjusting the anchor point.
  • Apply named materials to each group of objects that will share the same visual appearance in Unity. The material groupings do not need to be the same as the animation groupings.
