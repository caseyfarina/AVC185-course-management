Unity Scripts

Send An Event

  • _collectionManager.cs
    • Manage and track a collectable. Events can be sent to the "increment" and "decrement" functions
  • _keyEventOnDown.cs
    • send an unity event when pressing a key on the keyboard
  • _countDownKeyEvent.cs
    • send a unity event when pressing a key and limit the number of events
    • Display the number with a TextMeshPro object
  • _onTriggerEnterEvent2D.cs
    • send a unity event when entering a 2D collider set to trigger
  • _clickEventOnCollider.cs
    • send a unity event when clicking on an object with a collider
  • _onCollideThresholdEvent2D.cs  
    • send a Unity event when a 2D collider hits another 2D collider, set a threshold below which an event will not be sent
  • _onTriggerStayEvent2D.cs
    • send a Unity event when a 2D collider stays in contact with a 2D trigger for a given number of frames
  • _destroyObject2DTrigger.cs
    • destroy objects with a collider that come in contact with this 2D trigger
  • _onTriggerEnterEvent3D.cs 
    • send a unity event when entering a 3D trigger
  • _onTriggerEnterEventTag3D.cs 
    • send a unity event when an object with a specific tag enters a 3D trigger 
  • _onTriggerExitEventTag3D.cs
    • send a unity event when an object with a specific tag exits a 3D trigger
  • _countdownTimer.cs
    • Count down in time and send an event once time runs out
    • The time can be displayed in a TextMeshPro Object
  • _scoreManagerEvent.cs
    • increment an accumulating score via an event.
    • The score can be displayed in a TextMeshPro Object

Receive an Event and Do Something

  • _spawnSingleObject.cs
    • spawn a single prefab object

Movement

  • _lateralMovementController.cs
    • move an object left and right (x axis) with a min and max clamping value
  • linearMovement.cs
    • Add to an object to move at a consistent speed. Variation between objects can be increase with the "random percentage" slider.
  • linearRotation.cs
      • Add to an object to rotate at a consistent speed. Variation between objects can be increase with the "random percentage" slider.

Instantiating or Creating/Spawning Objects

  • simpleSpawner.cs- Add to an object to spawn a prefab. The time between spawns can be randomized.
  • timeDestroy.cs - Add to an object to automatically destroy that object after an elapsed amount of time. 

Shaders
