Project Objective: Students will animate the hero character for a third person 3d game while focusing on the principles of animation as applied to idle, run, and jump animations.

Course Competencies

    • Navigate Program Interface and tools.
    • Use animation techniques to create linear experiences
    • Use principles of animation
    • Create non-linear experiences

Project Requirements:

  • Process:
    • Make the indicated changes to topology and texturing of your character if necessary. Consider individual and group feedback when making your decisions. 
  • Requirements
    • There should be three animations that exhibit attention to detail and principles of animation. 
      • Idle 
      • Run
      • Jump

Deliverable:

  • Canvas

      • All three C4d animation files
        • Run
        • Jump
        • Idle
      • A screen recording video where you discuss the following
