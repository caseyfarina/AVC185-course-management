Project Objective: Learn about the UV unwrapping and 3D workflows for detailed asset design.

Course Objectives;

    • Navigate Program Interface and tools.
    • Create storyboards, develop concepts, visual style
    • Manipulate text and related objects, variables and expressions

Project

  • Build a modular platform that fits the theme of your game
    • Model the asset in C4D
    • UV unwrap the asset in C4D
    • export as a properly formatted FBX for Substance Painter
    • Texture the object in Substance Painter
    • Export the SP textures as a Unity URP SBSAR file
    • Import the SBSAR and the FBX into Unity and use the platform asset in your game 
  • Build, model, and texture 2 other objects for your game. 
    • NOT the main character, we're getting to that
    • Consider redesigning an asset you already built that needs revisions to fully realize your vision. 

Deliverable: 

  • Use OBS to give us a tour or your new objects 
    • Show each object in C4D and be sure to show the UV map
    • Show each object in SP and discuss how you decided to paint the textures
    • Show each object in Unity and discuss how you implemented it and how it can be used as a modular design component. 
    • Keep the recording to less than four minutes
    • Mp4 format 1920 1080
