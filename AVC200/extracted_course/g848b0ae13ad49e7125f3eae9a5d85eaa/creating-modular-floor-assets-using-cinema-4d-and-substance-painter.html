Create your own set of modular tiles to use as the floor in your 3D game.  Add some smaller details (to start establishing the world in your game).

First tile set

  • Straight path
  • Turn path
  • Intersection path
  • Blank path

Second tile set

  • four different tiles of your own design that integrate with the four original tiles. What other tiles can you imagine needing as your begin to layout the level of your 3D game? (path termination? power up? ) You are required to make some of your own alphas and textures that you could use in substance. Use these in your tile designs. 

Deliverable:

A screen recording (OBS of the unity editor) showing off all eight of your tile designs. Demonstrate that all eight will tile seamlessly.
