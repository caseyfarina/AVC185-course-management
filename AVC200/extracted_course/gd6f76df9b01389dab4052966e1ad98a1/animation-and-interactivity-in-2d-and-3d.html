The Principles of Motion Design will guide our animation throughout the semester.

Project Objective: Students will build two scenes that exhibit animation triggered by interactivity. All of the animation will draw from the Principles of Motion Design.

Course Objectives

    • Use principles of animation
    • Create non-linear experiences
    • Create and use advanced buttons for user input
    • Manipulate text and related objects, variables and expressions
    • Create movie clips, navigate timelines

Project: Create two scenes in Unity that demonstrate interactivity and the Principles of Animation. Use the same project from week one. Just create new scenes in that project.

Scene One: 2D

  • Mouse click for bouncing ball with anticipation
    • Use the UI to label the animation
  • Button press for spinning ball with anticipation
    • Use the UI to label the animation
  • Looping Linear Rotation
    • Use the UI to label the animation

Scene Two: 3D

  • Mouse Click for spinning Capsule with follow through/overshoot
    • Use the UI to label the animation
  • Mouse Click for Bouncing Sphere(3D) with squash and stretch
    • Use the UI to label the animation
  • Applied Learning: Design your own animation that draws on the Principles of Motion Design that is triggered by the user
    • Use the UI to label the animation
