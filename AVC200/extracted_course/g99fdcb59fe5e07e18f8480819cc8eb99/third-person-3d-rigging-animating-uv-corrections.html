Project: Correct the UV unwrapping and painting problems from last week then adjust the character leg rig and animate the legs in a walk or run cycle.

  • Make the appropriate changes to your character model based on your feedback and feedback from other students
    • "Save Incremental" so a trail of your previous versions exist incase you need to go back
  • When reimporting your mesh into Substance Painter, you do not need to start from scratch, rather update the mesh file in the project configuration.
  • Use the triplanar projection in substance painter to overlay a straight pattern over curved UV islands.
  • Use Fill Layers with masks as the base paint for the character and add details on a Paint layer.
  • Adjust the IK rig for the feet
    • Remove the existing track to modifier on the foot bone
    • extrude a foot target from the toe 
    • Split the foot target from the armature
    • use the child of constraint to make the foot target a child of the leg controller
    • Add the damped track to constraint to the foot bone and set the foot target as the target of the damped track to constraint
    • Adjust the appearance of the foot controller bone to a circle or cube
  • Save a version of the file as "complete rig"
  • Save a new version of the file as "walk cycle"
    • Animate the legs in a walk or run cycle
