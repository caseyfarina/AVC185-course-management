Please reformat your 3D models designed in C4D for Unity according to best practices. 

Export Checklist

  • Use the "connect objects" command https://help.maxon.net/c4d/en-us/#html/5666.html?Highlight=connect%20objects
  • Adjust the angle of the Phong Tag for correct shading https://help.maxon.net/c4d/en-us/#html/TPHONG.html?Highlight=phong%20tag
  • Adjust the anchor point. You can precisely place the anchor point with the "axis center" tool. https://help.maxon.net/c4d/en-us/#html/TPHONG.html?Highlight=phong%20tag
  • Select all of the polygons in polygon mode to check the polygon normals. Use the "align normals" command. https://help.maxon.net/c4d/en-us/#html/5669.html?Highlight=align%20normals
    • The outside of the mesh should be orange, not blue. 
    • https://help.maxon.net/c4d/en-us/#html/5669.html?Highlight=align%20normals

Download the Triplanar shader from the Important Unity Scripts/Shaders Resource. Use this shader to apply materials to your objects without texture distortion. Only use this shader on objects that do not animate. 

https://youtu.be/ccs9DQTW6yk?t=3138

You can find texture to use for your objects at the following. If there are optional versions of a normal map available, please download the OpenGL Normal map.

  • https://polyhaven.com/textures
  • https://ambientcg.com/list
