Project Objective: Students will model, UV unwrap, texture, and deploy a 3D object in a game environment.

Course  Objectives:

  • Use animation techniques to create linear experiences
  • Optimize performance
  • Manipulate text and related objects, variables and expressions
  • Use principles of animation 

Workflow:

  1. Model the following objects
    1. A cube with extrusions that form a platform
    2. A sphere with extrusions that create an outline on the surface
    3. A tube or Cylinder with extrusions that create a lip at the top and bottom. 
  2. UV unwrap all the objects in Cinema 4d
  3. Export the objects as individual FBX files
  4. Paint each object in Substance Painter
    1. Use at least two different materials and a mask
  5. Export textures from Substance Painter
  6. Import the FBX files and textures into Unity
    1. Use your existing Unity project but create a new scene.
  7. Create squash and stretch and overshoot animations for each object that are triggered when the objects are clicked.
