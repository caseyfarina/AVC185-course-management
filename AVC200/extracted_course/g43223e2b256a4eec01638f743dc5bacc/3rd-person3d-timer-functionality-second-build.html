Project Objective: Students will add conditional logic and timer logic to a 3D game while refining sound design, level design, 3D models, UI, and VFX design.

  • Course Competencies
    • Create storyboards, develop concepts, visual style
    • Use principles of animation
    • Create non-linear experiences
    • Create and use advanced buttons for user input
    • Manipulate text and related objects, variables and expressions
    • Use sound in production
    • Optimize performance
    • Distribute various output formats

Project Requirements:

  • Process:
    • UI
      • Select a Font that will fit the theme of your game
        • Be conservative, let the imagery be the focus
        • https://fonts.google.com/
        • Do not place type elements up against the edge of the screen 
    • Game Timer
      • Start line
        • Start the timer
        • Clear the Welcome/Instruction Message
      • Bonus Time Powerups
        • Add bonus time powerups throughout the level that make it possible for the player add more time as they clear each section
        • You can control the amount of powerup time in the timer script
        • Particle burst when Bonus time is acquired
        • Sound Effect when bonus time is acquired
      • Finish Line
        • Win Music
        • Win Message
        • Win Particles
      • Time Out
        • Lose message
        • Freeze Player
        • Restart Script 
        • Restart Message
    • Modeling
      • Add a start line and finish line area that fits with the theme of your game
    • VFX
      • Refine the particle effects you added last week based on feedback
    • Sound
      • Balance the volume of the music and sound effects

Deliverable:

  • Canvas
    • A zip file containing the following
      • An OBS recording of you playing the built version of your game. Demonstrate what has changed and how you implemented the timer and bonus time powerups in your game.
        • Limit to three minutes
    • A link to the google drive build of the game
  • Slack
    • Post your screen recording and comment on at least three other student posts.
