Project Objective: Students will edit an animated character and import that character into a game engine.

  • Course Competencies
    • Navigate Program Interface and tools.
    • Use animation techniques to create linear experiences
    • Create storyboards, develop concepts, visual style
    • Use principles of animation
    • Create non-linear experiences
    • Create and use advanced buttons for user input

Project Requirements:

  • Process:
    • Adjust your rigging, modeling and painting as indicated in the feedback
    • Export you animations in fbx and import into Unity
    • Configure your character animations in Unity to properly animate the character according to player input. 
      • Your main character should
        • idle when standing still
        • run while moving
        • jump when the jump key is pressed
    • Import you textures from substance painter and apply the textures to the character

Deliverable:

  • Canvas
    • A zip file containing the following
      • A screen recoding using OBS of you controlling your animated character in the Unity editor. While demonstrating the character animation please discuss the following
        • What would you do differently now that you've modeled, rigged, painted, animated, and implemented a game character?
        • What part of the process did you like best? worst? 
        • How does the character appearance and animation relate to the theme of your game?
