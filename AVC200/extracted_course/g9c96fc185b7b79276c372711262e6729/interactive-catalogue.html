This week  we're creating an interactive catalogue that will showcase our 3D modeling and design skills. This project will involve working with typography and using a design grid. Please review the DMA typography guidelines in the following information. You may ONLY use fonts from the provided zip. 

General Typography Practices

The FUTUR ( <-- an amazing resource for design and business knowledge, click link) has put together a great manual for typography. Please download and take notes. Follow these guidelines.

Typography-Manual.pdf  

The animated version of the manual.

Allowed Fonts for this project and All other projects:

I have put together a zip file containing all of the fonts you may use during the semester. These are opensource (free to use) versions of the fonts in the FUTUR typography manual. If you have any of the fonts listed in the FUTUR manual you may use those, or the Google Font equivalents that I've included in this zip file. You may not use other fonts. Beginning students often attempt to "experiment" with font selection and make design decisions that would disqualify their otherwise good work from consideration in job searches, contest, and competitions.

Requirements

All of the following objects should be selectable with descriptions

  • The Glendale Community College Palm Tree (use the logo object ai workflow). You can download the GCC logo here https://www.maricopa.edu/about/brand-guidelines
    • You'll need to open the EPS file in Illustrator and save it as an AI file before you drag it into Cinema 4D.
    • https://helpx.adobe.com/illustrator/using/importing-eps-dcs-autocad-files.html

  • Vase
  • Lava Lamp
  • Neon Sign
  • One Lathe object of your own design
  • One extrude object of your own design
  • 3 new objects (your own choosing) created using a volume modeling workflow
  • 3 other objects modeled with any technique (think about combining techniques)
