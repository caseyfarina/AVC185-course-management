Project Objective: Students will apply texturing skills and use scripts to create collectable coin mechanic in a game.

Course Competencies Covered:

  • Use principles of animation (IV)
  • Create non-linear experiences (V)
  • Create and use advanced buttons for user input (VI)
  • Create movie clips, navigate timelines (VII)
  • Manipulate text and related objects, variables and expressions (VIII)

Project Requirements:

    • Model a coin and paint it in Substance Painter
      • The collectable script is on the Important Unity Scripts/Shaders Resource 
      • export the textures as an sbsar file from Substance Painter and import into Unity
      • export the mesh as an fbx from Substance Painter and import into Unity
    • Set up a collectable mechanic in a new room that allows the player to acquire coins.
      • Use Unity UI to communicate to the player how many coins they've collected
      • Send a message to the player once they've collected four or more coins

Deliverable:

  • A video demonstrating the player collecting four coins and receiving a message.
