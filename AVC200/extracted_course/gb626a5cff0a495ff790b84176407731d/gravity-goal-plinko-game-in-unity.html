Project Objective: Students will build the foundations for a 2D Plinko game in Unity. The basic game loop and visual components will be established.

Course Objectives

    • Navigate Program Interface and tools
    • Use principles of animation
    • Create non-linear experiences
    • Manipulate text and related objects, variables and expressions

Project: these requirements are minimums. Please feel free to add more to your game at your discretion. 

    • sprite sheet - I drew the sprites in Adobe Illustrator. You may use illustrator or any program that will export a png with an alpha channel
      • Please give the game your own visual design, do not copy the design of the example sprites
    • four different stationary obstacles
    • two different animated obstacles - experiment with animating both position and rotation
    • 1 - 3 goal buckets
    • 2D destroy trigger to clean up balls that do not win
    • Moving ball spawner - the example moved side to side. Are there more possibilities?
    • User Interface UI
      • Animated start message
      • animate win message
      • animated lose message
  • Fonts you should use for good design practice
    • GCC DMA FONTS.zip 
      • How to install all the fonts at the same time https://www.youtube.com/watch?v=y0gJyyrPpsc
    • The Futur Typography manual:
      • https://thefutur.com/content/typography-manual
  • All scripts are available in the Important Unity Scripts/Shaders Resource
    • This is where all the scripts will be stored

Deliverable:

A recording of your game that satisfies the following requirements:

  • show two wins
  • show two losses
  • 1280 x 720
  • mp4 file (30 or 60 fps)
