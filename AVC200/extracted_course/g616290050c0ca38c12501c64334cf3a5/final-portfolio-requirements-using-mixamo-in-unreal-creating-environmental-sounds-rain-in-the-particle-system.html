Project Objective: Students will design, model, and texture game ready particle effects, NPC's, and start assembling a portfolio.

  • Course Competencies
    • Navigate Program Interface and tools.
    • Use principles of animation
    • Create non-linear experiences
    • Create movie clips, navigate timelines
    • Optimize performance
    • Distribute various output formats

Project Requirements:

  • 1st person Unreal Engine Game Requirements
    • Game Collectables
      • must include sound FX and particle FX
    • >= (greater than or equal to) - modeled and textured game assets that you designed
    • >= 3 Environmental Sound FX
    • >= Substance that you've created in Substance Sampler
    • At least one NPC that either modeled and Rigged in Mixamo, or downloaded from Mixamo and substantially retextured in Substance Painter
    • >= trigger zones that display game play information to the player with the UI. How could these messages encourage the player to explore the world?
    • >= 3 doors
    • Background music and ambience
  • Build the Unreal game and post it on your google drive and please set the link permissions to "anyone with link." 
  • Post three of your game props (the best objects that you modeled and painted, do not use the ball) from the Unity game on Sketchfab. Set the permissions on Sketchfab to "free" to not run into upload limits
  • Post three of your game props (the best objects that you modeled and painted), from the Unreal game on Sketchfab. Set the permissions on Sketchfab to "free" to not run into upload limits.
  • Start to configure a Behance portfolio page for the Unity game. Embed all three Unity game props from SketchFab
  • Start to configure a Behance portfolio for the Unreal game. Embed all three Unreal game props from SketchFab

Deliverable:

  • Canvas
    • A presentation of your completed Unreal game (< 4 minutes). Show us your favorite parts of the game and discuss what you would like to work on in the future.  You should record the built version of the game, not the game in editor. Please show all the new additions to the game. 
    • The link to the build game zip on google drive
    • The link to the Unity Behance page
    • The link to the Unreal Behance page
  • Discord
    • Please post the google drive link to your completed built game
