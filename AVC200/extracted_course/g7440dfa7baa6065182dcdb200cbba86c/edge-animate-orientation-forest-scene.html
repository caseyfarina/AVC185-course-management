Introduction

Over the course of the semester we'll be combining the concepts of visual representation that we learn from Understanding Comics and implementing them in Adobe Edge Animate. In order to do this, we need to become comfortable using Edge Animate (EA).

Reference

In order to complete this project you need to watch and take notes on Chapters 1 and 2 in the Lynda.com Series Edge Animate Essential Training with Chis Converse.

Project

Please create your own forest scene in EA that uses the following elements. You may use elements beyond what is listed below, but your scene must include at least these elements. Please make sure the stage is set to 1000 pixels by 700 pixels.

  • A rectangle with asymmetrically rounded corners
  • An ellipse that is clipped using the clipping tool
  • Another ellipse that is colored using the gradient tool
  • Use a text box and one of the Adobe CC Fonts to create a title for you scene. The title should be "A Forest of Interactivity"
  • Three elements you created in Illustrator (probably objects)
  • Two elements you created in Photoshop (probably textures)

Your scene should adhere to the basic rules of design and look aesthetically pleasing. Be sure to properly format and design your type. Review the use of grids in type layout by watching the Lynda.com Series Foundations of Typography: Working with Grids with Ina Saltz.

I would strongly suggest generating or selecting a color palette from KULER.

Deliverable

Please turn in a ZIP file of your entire project folder and a 200 word document that explains how your scene meets the requirements of the project. Also discuss any problems you had and ideas for elements that you would like to include in the future. PLEASE DO NOT SUBMIT WPS FILES FOR YOUR DOCUMENT. Acceptable file formats are .doc, .rtf, .txt, .docx.

  • ZIP of your project folder
  • Word doc discussing how your forest project meets the requirements of the project
