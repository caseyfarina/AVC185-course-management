ROLL-A-BALL GAME REQUIREMENTS

  1. 6 animated prefab obstacles
    • Use this script to randomize animation time on a prefab AnimationSpeed.cs
  2. 6 animated prefab decorative objects
    • Use this script to randomize animation time on a prefab AnimationSpeed.cs
  3. Create a new design for the pickups, they should be animated and attract attention. Use this script to change the total number of pickups needed to win the game. PlayerControllerEdit.cs
  4. Bring in at least 5 different textures to use on materials
  5. Generate Normal Maps for those 5 different textures
  6. Create a Post Processing Profile and add at least AMBIENT OCCLUSION
    • A good resource for the post processing profile https://github.com/Unity-Technologies/PostProcessing/wiki/Quick-start
