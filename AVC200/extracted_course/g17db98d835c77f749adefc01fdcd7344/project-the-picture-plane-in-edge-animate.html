The "picture plane" in Understanding Comics has three corners: reality, icon, and abstraction.

  • Reality: the image looks similar to how we experience it everyday experience
  • Icon: the image is a greatly simplified version that retains enough elements to still be identifiable as the object.
  • Abstraction: the image has been broken down into component lines and shapes, and does not retain the identity of the original image.

Your project is to create an Edge Animate project that allows the user to explore the "picture plane." The patch should take user input (mouse, button, etc.) and use that to explore the same subject as it is represented on the three corners of the "picture plane." Please watch the tutorial for an example project.

You will need to address several issues.

  1. How will your patch communicate to the user how it functions?
  2. How will your patch transition the imagery between the three different corners of the picture plane?
  3. What subject will you select to represent with three different versions?

Deliverable:

A zip file containing the edge animate project folder, and a 200 word description of how you put this project together and how you overcame your creative and technical obstacles.
