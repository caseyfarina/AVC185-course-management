Instruction:

  • Unity Scripts, Using Unity Events to create puzzle functionality, Unity User Interface (UI)
    • https://www.youtube.com/watch?v=Agu--Uo8QYM
  • Feedback and Introduction to Substance Samper
    • https://www.youtube.com/watch?v=VMC8FZd-Mek

Project Objective: Students will apply texturing skills and use scripts to create aa puzzle experience.

Course Competencies Covered:

  • Use principles of animation (IV)
  • Create non-linear experiences (V)
  • Create and use advanced buttons for user input (VI)
  • Create movie clips, navigate timelines (VII)
  • Manipulate text and related objects, variables and expressions (VIII)
  • Use sound in production (IX)

Project Requirements:

  • Create at least four materials using Substance Sampler, import them into your Unity project and apply them using Triplanar projection when necessary.
    • Use an image you took to create 2 substances
      • Avoid shiny surfaces, they will not work well for this process.
    • Create 2 more substances using a Substance Sampler base material and alter it using some of the Substance Sampler filters
      • https://helpx.adobe.com/substance-3d-sampler/filters/wear-and-finish.html
      • Do not use the gum generator
  • Create a NEW room or space in your project that has a multistep puzzle for the player to solve.
    • The puzzle must include at least 2 UI messages
      • Import a font from google fonts to use in your project. Do not use the Text Mesh Pro default font.
    • The puzzle must include a locked door that solving the puzzle unlocks
    • There must be a visual indicator that the door is locked, and then unlocked (red and green light)
    • Model at least two different rocks to include in this new space. They can be used in the puzzle or as a prop.
    • Ask yourself if someone who is new to this game would understand how the puzzle works? Should you include more written directions or less?

Revisions:

  • Implement any changes and suggestions from your feedback.

Deliverable:

  • A presentation video (less than 3 minutes) detailing:
    • Demonstrate the functionality of your new puzzle room
    • Explain how you designed your puzzle, and which scripts you used.
    • Show how you used each of the four new Substances from Substance Sampler in your level.
    • Any questions or feedback?
