Set up the following in the Avoidance Game.

  • 3 enemies
    • One Image Sequence enemy animated in After Effects
    • One Enemy animated in Unity
    • One Enemy the uses the "Apply Root Motion" checkbox to move at a nonlinear rate across the screen
  • A hero with Idle, left, right, up, and down animations
  • Remember to not use any existing intellectual property (Mario, Zelda, Avengers, etc).
  • Try taking a simple concept and twisting it slightly  (a banana avoid vegetables .......... in space) for example
  • A final goal that puts the game in a win state
  • When the player contacts the enemies the game enters the "lose" state
