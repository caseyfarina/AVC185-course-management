Project Objectives: Students will create a game design document and begin to prototype a 3rd Person 3D game. 

Course Objectives:

  • Navigate Program Interface and tools.
  • Create storyboards, develop concepts, visual style
  • Create non-linear experiences
  • Create movie clips, navigate timelines
  • Optimize performance

Project: 

  1. Create a one page game design document to plan out your 3rd Person Game
    • All games will use the Unity Template I introduced this week
    • Your design document should have the following
      • A brief description of the game world. Three sentences describing the world your game exists in
      • A detailed map of the level
        • The level needs to have at least four rooms or zones
        • You can do this in Blender, or with paper, or in Lego, whatever you're most comfortable with
      • A description of how the obstacles/puzzles function in each zone
      • A color palette
      • material palette
      • A sketch of the main character
      • A description of each zone and the mechanics of that zone
    • Your design document should be a large image or a pdf. 

Deliverable:

  • Canvas
    • Your game design document as an image or pdf (if you submit an image make sure the resolution is high enough to read all the text)
    • A < 3 minute discussion of your game design document
  • Discord
    • Please post your game design document and comment on at least three other student posts
