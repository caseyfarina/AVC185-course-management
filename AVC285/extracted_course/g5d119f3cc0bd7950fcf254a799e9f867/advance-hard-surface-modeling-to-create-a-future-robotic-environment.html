We are going to take a deep dive into hard surface modeling techniques in order to create our own kit bash library to construct a robotic, futuristic, or mechanical scene.

AESTHETIC

Kit Bash = The art of constructing original forms from a library of smaller parts and components. This name of this technique comes from the special FX practice of creating models from parts of other model "kits." The tiny parts and components are often referred to as "Greeblies."

Why?

  • To more easily create an environment that is thematically connected (built from the same parts).
  • To more easily add detail and thus SCALE

 This concept can easily be applied to 3D modeling.

If the tool is not is the same location in the interface you can SHIFT C to search to the tool. The only tool that has changed names is "set selection" is now "store selection." Even if you use different tools, make sure the topology is the same.
