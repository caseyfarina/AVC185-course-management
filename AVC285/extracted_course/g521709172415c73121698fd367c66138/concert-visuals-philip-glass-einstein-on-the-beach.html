Scripts:

  • ForceRenderRate.cs
    • this script will force unity to render at a specific frame rate. We want the project to look cinematic so set it to 24 FPS
  • AnimationSpeed.cs
    • Put this script on animated objects to set a range for their animation speed. Each object that has this script attached will randomly select a unique speed within the range. You can easily create variation by copying and pasting prefabs.
  • cameraControls.cs
    • Put this script on  blank object and drag the virtual cameras to the different key control slots
  • toggleState.cs
    • put this script on your CONTROLS object and use it to assign keys to your different objects you want to trigger
  • toggleRotation.cs
    • Put this object on your CONTROLS object to turn on and off object rotation
  • toggleNoise.cs
  • Perlin.cs
    • Put both of these scripts in your project, but only use the toggleNoise to move your objects around with noise
  • toggleOrbit.cs
    • use this script to orbit one object around another
