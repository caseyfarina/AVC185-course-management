Example File:

tentacle project.zip 

 

Please download and inspect my project files. Pay particular attention to the RS render settings. Save the RS render settings present on your machine so you can quickly render the first draft of the Simulation Reel. 

Simulation Reel: Choose at least 5 scenes from the previously simulation assignments to improve and render in Red Shift. Edit these scenes into a reel.

  • Modeling - Does the scene need other elements to work? What would make this visually and artistically interesting?
    • Would any of the elements benefit from UV unwrapping and texturing in Ssubstance Painter. Try to unwrap and paint at least one object.
  • Simulation  - Clean up the simulation and consider increasing the iterations and sub-steps if the simulation is too "springy"
    • https://help.maxon.net/c4d/en-us/#html/OPBDSCENE-PBDSCENE_GROUP_XPBDPROPERTIES.html?Highlight=iterationshttps://help.maxon.net/c4d/en-us/#html/OPBDSCENE-PBDSCENE_GROUP_XPBDPROPERTIES.html?Highlight=iterations
    • Cache simulations before rendering !
  • Cinematography 
    • Should there be camera movement?
      • If so Please set one keyframe at the beginning of the scene and one at the end
      • Please set camera keyframe interpolation to Linear
    • Choose a focal length
      • 35 mm = the C4D default - if you keep this be able to answer... Why?
      • 55 mm =  the focal length of the human eye.
      • 120 - 240 mm = isometric/telephoto focal lengths. This will require moving the camerra further away from the scene, however, the scene will look more isometric
  • Lighting
    • Look at the example file
      • Key Light
      • Fill Light
      • Dome Light with HDR. How does your HDR selection effect the reflections in the scene?
  • Materials and Textures
    • Use some of the preset RS materials from the asset browser
    • Create some of your own materials using the RS Standard Material
      • Using some of the Scratch textures in the Asset browser on the bump and roughness channels can be very effective
