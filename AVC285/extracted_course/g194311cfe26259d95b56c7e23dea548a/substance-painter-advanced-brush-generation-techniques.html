Assignment Overview:

This assignment focuses on the procedural creation of multi-layered brushes for Substance Painter using Blender. The objective is to develop brushes that can be dynamically adjusted and applied in Substance Painter to enhance textural detail and complexity in 3D models.

Instruction:

Before you begin, familiarize yourself with the following advanced techniques in Substance Painter:

  • Using the cloner to create a procedural alpha generator,
    • Using the multi-pass renderer in Blender to generate Substance brush layers:
    • https://caseyfarina.github.io/lecture-redirects/?class=avc285&lecture=week11-lecture1
  • Substance Painter project review,
    • Using mesh maps to create color variation
    • Using the mask outline effect in conjunction with anchor points
    • https://caseyfarina.github.io/lecture-redirects/?class=avc285&lecture=week11-lecture2f
  • Additional Anchor Point Examples
    • https://www.youtube.com/watch?v=944ci1laePI

Project Objective:

Revise and reimagine your Mat painting by adding the following and improving on last week's design. 

  • Add at least two procedural brushes that include an
    • alpha
    • emissive (ID map)
    • Height map
    • Use the Baked ID map to drive several layer mask
  • Vary the base color across the surface of your project by using a mesh map color overlay
  • Use the "mask outline" effect at least once
  • How is this Mat an improvement on last week's design?

Course Objectives:

By completing this project, you will achieve the ability to:

  • Apply material to surfaces and 3D models using various painting and mapping
    techniques. (I)
  • Determine illumination effects on a 3D model. (II)
  • Apply various deformation principles for animation and special effects. (II)
  • Produce a 3D character study.
