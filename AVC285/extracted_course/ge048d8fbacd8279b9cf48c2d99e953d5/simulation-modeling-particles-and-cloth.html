Create an mp4 with the following scenes. Please use the viewport renderer when possible:

  1. Copy the table, chairs and place settings model from ART 185 into a new scene. Put a table cloth on the table. You may want to use a simplified table top as the cloth collider. If you have time try to add napkins to the scene.
  2. Create a flag waving in the breeze for at least five seconds.
  3. Create a simple machine using the kitbash parts in the asset browser (maybe 5 - 10 pieces), Animate that machine falling apart using the Fracture object.
  4. Use a cloner to create a tower of Jenga blocks and animate it falling down when a block is removed. 
  5. Create some drapes for a window. Use the Symmetry object to make two sides.
  6. Create a machine that throws objects against a wall or into a ball pit. Perhaps a foam pit? Have fun with it.  

Deliverable:

An mp4 of all the scenes edited together. Please upload to Canvas and post on Slack.
