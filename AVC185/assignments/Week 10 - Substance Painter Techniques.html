Week 10: Substance Painter Techniques

Instruction:

  • Base Layers In Substance Painter
    • AVC185 Week 10 Lecture 1

Course Competencies:

  • 5. Execute finished models by employing materials, environmental effects, and rendering

Project Objective:

Apply advanced Substance Painter techniques including smart masks, anchor points, and translucency for professional material creation.

Project:

  1. Paint the class Oil Lamp using the following
    1. Smart masks
    2. Mesh Maps
    3. Anchor Points
    4. Translucency
    5. Import a warning label
  2. Model a lamp of your own choosing to scale from reference.
    1. Import the reference into Blender and model to scale
    2. UV Unwrap the lamp in Blender
  3. Paint your lamp in Substance Painter using the following
    1. Smart masks
    2. Mesh Maps
    3. Anchor Points
    4. Translucency
    5. Import a warning label
