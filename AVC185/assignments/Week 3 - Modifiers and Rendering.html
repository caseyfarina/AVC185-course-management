Project Objective:

Implement modifier-based workflows and rendering techniques to create complex playground scenes with proper materials and lighting.

Project:

Model the following objects in a single Blender Scene. Organize each object under a blank axis and in a Blender Collection.

  1. Swing Set
    1. Use the array modifier and the bezier curve object to create a chain with the appropriate density of topology.
    2. Use SubD surface modeling and the Solidify modifier to create the swing seat.
  2. APPLIED LEARNING construct one additional playground element (use a reference image) and "connecting" playground elements (bridges, platforms, etc.)
  3. Apply materials to each object
    1. label all materials
    2. Use the Assign function to apply materials to a subset of the object topology. 
  4. Render a png of the playground from a view that displays the entire playground containing
    1. Swing set
    2. Slide
    3. Thuderdome
    4. Merry-Go-Round
    5. Your Object
    6. Connecting structures
