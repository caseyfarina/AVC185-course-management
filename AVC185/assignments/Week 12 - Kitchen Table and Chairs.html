Project: Model and Paint a table and chair to scale.

  • Model, UV unwrap, and paint the table and chairs in Substance Painter.
  • Calibrate both objects so they are modeled to scale
  • Double check that you did not create any additional UV layers BEFORE exporting to Substance Painter
  • Pay particular attention to the UV unwrapping to insure the wood grain is handles correctly in Substance Painter.
  • Create two different versions in substance painter If you use some smart materials edit them to adjust some of the smart masks to add some paint detail.
    1. Wood table and chairs
    2. Wood with peeling paint table and chairs
  • Use Cycles to create STILL IMAGES of the chairs, tables, and lamp scene
