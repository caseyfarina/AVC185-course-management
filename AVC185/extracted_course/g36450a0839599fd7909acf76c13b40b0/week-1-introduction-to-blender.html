  • 1. Apply basic geometry principles for construction of 3D models

  • 6. Apply basic animation principles to models

Project Objective:

Create an abstract 3D scene using primitive objects to demonstrate fundamental Blender interface skills and basic modeling workflow.

Project:

Create an AVC185 Week 1 Lecture 1(not representational) scene in Blender that uses the following 

  • At least 30 3D primitives. Use at least one of each of the following
    • UV Sphere
    • Cylinder
    • Cone
    • Ico Sphere
    • Torus
  • At least three point lights of various intensities
  • Place camera at the ideal location to view the scene
  • Use the following tools during the scene's construction
    • Move
    • Rotate
    • Scale
    • shift + transform (nudge)
    • ctrl + transform (snap)
  • Enable smooth shading on the objects you would like to appear smooth
  • Properly manage the topology so the primitives have an appropriate density of faces/polygons
  • Use orthographic projections to precisely place objects in space
  • Apply materials to the objects to create a compelling abstract image
    • You may use materials that you configure 
    • or simple material from Blender Kit
