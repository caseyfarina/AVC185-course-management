Project Objective:

Create an abstract 3D scene using primitive objects to demonstrate fundamental Blender interface skills and basic modeling workflow.

Project:

Model to scale from reference, UV unwrap, and paint two sets of silverware: One standard set, and one fancy set

  • Instruction
    • Fork and spoon using the demonstrated Subdivision Surface workflow. Pay particular attention topology required to increase or decrease face density when using a Subdivision surface workflow
  • Applied Learning (Make the best modeling decisions using all of the techniques you've learned)
    • knife, and spoon
  • Paint all of the silverware in Substance Painter and use the height channel to create additional surface features
    • Use lazy mouse
    • Use paint Symmetry
    • Use blur effects to smooth out the height channels
