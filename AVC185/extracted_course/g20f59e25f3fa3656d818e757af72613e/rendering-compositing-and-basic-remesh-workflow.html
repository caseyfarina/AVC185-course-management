Objective: Students will create an animated showcase for a series of 3D models.

Course Competencies;

  • Apply basic geometry principles for construction of 3-D models
  • Create 3-D shapes using linear interpolation and polygonal construction
  • Demonstrate ability to modify shape appearance through the use of lights and camera angles. (III)
  • Execute finished models by employing materials, environmental effects, and rendering. (III)
  • Apply basic animation principles to models. (IV)

Project:

Model a "climbing animal" for your playground. Execute the changes and improvements detailed in the feedback. Render a showcase movie of your playground

  1. Model a climbing animal, creature or object for your playground using the Primitive Blockout, Join, Remesh, Sculpt Workflow 
    •  Suggestions
  2. Apply a sand material (from Blender kit) to the playground floor and create a "box" box around the playground. Use displacement and the Multiresolution Modifier to displace the sand. 
  3. Execute all changes and improvements detailed in the class discussion. I will ask you to continually improve the projects throughout the semester. Making changes is part of design work, be prepared to continuously work on several projects.
  4. Render a showcase video from After Effects that contains the following 
    1. At least 4 still renders from the playground
      1. Slow zoom in
      2. Slight Vignette using an Adjustment Layer
    2. Use a text layer to create a title that fades in and out at the beginning of the video
    3. Render a three second animated camera shot of the playground using Render Kit
    4. Render the showcase movie from Media Encoder using the 1080 Youtube preset
