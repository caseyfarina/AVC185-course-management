Objective: Students will paint a detailed scale model using texture layers.

Course Competencies;

  • Apply basic geometry principles for construction of 3-D models
  • Create 3-D shapes using linear interpolation and polygonal construction

Project:

  1. Paint the class Oil Lamp using the following
    1. Smart masks
    2. Mesh Maps
    3. Anchor Points
    4. Translucency
    5. Import a warning label
  2. Model a lamp of your own choosing to scale from reference.
    1. Import the reference into Blender and model to scale
    2. UV Unwrap the lamp in Blender
  3. Paint your lamp in Substance Painter using the following
    1. Smart masks
    2. Mesh Maps
    3. Anchor Points
    4. Translucency
    5. Import a warning label
