  • 1. Apply basic geometry principles for construction of 3D models

  • 6. Apply basic animation principles to models

Project Objective:

Create an abstract 3D scene using primitive objects to demonstrate fundamental Blender interface skills and basic modeling workflow.

Project:

  1. Use the Jade Toad Substance Painter sample to demonstrate the following
    1. Three or more paint layers that use metalic substances to outline or highlight some of the toad features
    2. A fill layer with the following. Use Triplanar projection
      1. Color Channel: Noise and a gradient filter to create subtle variations in color across of the surface. 
      2. Roughness Channel: Noise and a Levels filter to create variation in shininess across the surface. 
      3. Height Channel: Noise and a Levels to create height texture across the surface
    3. A fill layer of a metal that is masked onto the sphere the toad sits on
  2. Export the climbing animal from Blender. Do not export the sculpted animal (turtle)
      • Apply the Modifiers and Scale on each part of the animal
      • Join all of the separate meshes together
      • Check that the normals are correct, (blue not red). 
      • If necessary apply the weld modifier to clean up and inconsistencies caused by the Bevel Modifiers
      • Export the animal as an FBX
      • Import into SP and check the "auto unwrap" option
    1. Bake the mesh maps and use a smart material to texture the climbing animal
    2. Add some paint on top of the smart material by using a paint layer
