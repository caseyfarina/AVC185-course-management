Week 11: Lamp Revisions

Instruction:

  • Substance Painter to Blender, Setting up glass in the Eevee Renderer, Editing a model showcase reel. 
    • AVC185 Week 11 Lecture 1

Course Competencies:

  • 1. Apply basic geometry principles for construction of 3D models
  • 2. Build models using one or more shapes and apply Boolean operations
  • 6. Apply basic animation principles to models

Project Objective:

Create an abstract 3D scene using primitive objects to demonstrate fundamental Blender interface skills and basic modeling workflow.

Project:

  1. Revise the your lamp and the class lamp.
    1. It may be necessary to remodel sections of the lamp design
    2. Feel free to revise to improve where needed
    3. After remodeling you can import the new mesh into the same Substance Painter file. It will be necessary to re-bake the mesh maps and re-define the mask areas, however, the layering will remain intact. 
  2. Revise the texturing for your lamp design
    1. Use an opacity channel to create holes in the design that would be difficult to model.
  3. Create a showcase video for both lamps that consists of the following passes 
    1. beauty pass
    2. flat pass
    3. topology (geo) pass
    4. beauty pass again
  4. Both videos should meet the following specifications
    1. 1080 x 1920 resolution
    2. 24 fps
    3.  linear rotation keyframes
    4. 3 - 4 seconds per pass
