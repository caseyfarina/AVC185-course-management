Objective: Students will create an animated showcase for a series of 3D models.

Course Competencies;

  • Apply basic geometry principles for construction of 3-D models
  • Create 3-D shapes using linear interpolation and polygonal construction
  • Demonstrate ability to modify shape appearance through the use of lights and camera angles. (III)
  • Execute finished models by employing materials, environmental effects, and rendering. (III)
  • Apply basic animation principles to models. (IV)

Project:

  1. Model a 2D wood climbing Pig or Cow using the Hard Surface Boolean Workflow. Follow the example closely.
    1. Enable the "Booltools" addon
    2. Use the Bezier curve to design the body and ear shapes
    3. Use the fill tool to create a cap on each extruded bezier curve
    4. Use the Bevel Modifier with "Harden Normals" enabled to define the shape with rounded corners
    5. Use the Boolean modifier via the Booltools addon to cut bolt holes into the structure 
    6. Reorder the modifiers to make sure the Boolean cuts are Beveled. 
  2. Model a climbing turtle using the sculpting workflow. Follow the example closely.
    1. Use Bezier Curves and Primitives to block out the turtle shape
    2. Convert each part to a mesh and join the parts into a single mesh object
    3. Use the remesh modifier to prepare the turtle for sculpting, then apply the remesh modifier
    4. Focus on the following brushes when sculpting: Draw, Smooth, Crease Polish, and Grab
    5. Use the R shortcut to set the remesh density when sculpting, Do not drop the remesh distance TOO LOW, this will cause a significant Blender slow down.  Use the ctrl R shurtcut to execute the sculpting remesh. 
  3. Create a leaf using a plane and an image of a leaf with an alpha channel, scatter a few of these leaves in part of the playground. Do not use a material from blender kit, rather find a leaf png and construct your own material.
  4. Create a large rock to spread around the playground. Create the material from the provided textures, do not use a rock material from blender kit. 
    1. Use the following textures to create a rock material. Use apply the appropriate texture to the appropriate channel.
      1. textures rock.zip
    2. Apply to a Icosphere with a Subdivision surface modifier. Move some of the points around on the icosphere to edit the rock shape. 
    3. Use the following settings for the normal map node
  5. Execute all changes and improvements detailed in the class discussion. I will ask you to continually improve the projects throughout the semester. Making changes is part of design work, be prepared to continuously work on several projects.
  6. Render a  new showcase video from After Effects that contains the following 
    1. At least 4 shots with a slowly animated camera 
      1. Please change the camera keyframes to LINEAR
      2. Each camera shot should move in a straight line, the camera should not change direction
      3. Each shot should be at least three seconds long
      4. Slight Vignette using an Adjustment Layer
    2. Show all parts of the playground and the new additions
    3. Use a text layer to create a title that fades in and out at the beginning of the video
    4. Render the showcase movie from Media Encoder using the 1080 Youtube preset
