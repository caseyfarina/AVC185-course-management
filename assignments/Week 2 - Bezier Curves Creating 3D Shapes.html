Objective: Students will gain familiarity with the Bezier curve modeling tools in Blender. Students will focus on manipulating the tool and curve settings to influence topology. 

Course Competencies;

  • Apply basic geometry principles for construction of 3-D models
  • Create 3-D shapes using linear interpolation and polygonal construction

Project:

Model the following objects in a single Blender Scene. Organize each object in a Blender Collection

  1. Slide 
    1. Use two Bezier curves
      1. one for the path
      2. one for the slide profile
    2. Build the support structure out of bezier curves
    3. Build the stairs using a cube and the array and bevel modifiers
  2. Playground "thunderdome"
    1. Edit an Icosphere to create the correct topology for the playground dome
    2. Use the bevel and wireframe modifiers to outline the topology
  3. APPLIED LEARNING (I did not make a tutorial for this, please apply your skills): Model a playground merry-go-round .
  4. Do not worry about materials, just focus on the topology and shape accuracy this week
